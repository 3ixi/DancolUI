API参考文档

Dancol UI Framework 完整的API参考手册

概述

Dancol UI Framework 提供了简洁而强大的API，让您能够轻松地创建美观的用户界面。框架主要包含以下几个核心模块：

  • Dancol.utils - 核心工具函数
  • Dancol.Theme - 主题管理系统
  • Dancol.Toast - Toast消息系统

全局对象

框架会在全局作用域中创建 Dancol 对象，您可以通过这个对象访问所有的API。

// 检查框架是否已加载
if (typeof Dancol !== 'undefined') {
    console.log('Dancol UI Framework 版本:', Dancol.version);
}

核心API

核心API提供了常用的DOM操作和工具函数。

Dancol.utils.$(selector, context)

Element | null $(selector: string, context?: Element)

选择单个DOM元素，类似于 document.querySelector。

参数
参数名 类型 必需 描述
selector string 必需 CSS选择器字符串
context Element 可选 搜索上下文，默认为document
示例
// 选择元素
const button = Dancol.utils.$('.my-button');
const input = Dancol.utils.$('input[type="text"]', form);

Dancol.utils.$$(selector, context)

NodeList $$(selector: string, context?: Element)

选择多个DOM元素，类似于 document.querySelectorAll。

参数
参数名 类型 必需 描述
selector string 必需 CSS选择器字符串
context Element 可选 搜索上下文，默认为document
示例
// 选择所有按钮
const buttons = Dancol.utils.$$('.dancol-btn');
buttons.forEach(btn => {
    console.log(btn.textContent);
});

Dancol.utils.on(element, event, handler, options)

void on(element: Element, event: string, handler: Function, options?: boolean | Object)

为元素添加事件监听器。

参数
参数名 类型 必需 描述
element Element 必需 目标DOM元素
event string 必需 事件类型
handler Function 必需 事件处理函数
options boolean | Object 可选 事件选项
示例
const button = Dancol.utils.$('#my-button');
Dancol.utils.on(button, 'click', function(e) {
    console.log('按钮被点击了');
});

主题系统

主题系统允许您动态切换应用的视觉主题。

Dancol.Theme.applyTheme(theme)

void applyTheme(theme: string)

应用指定的主题。

参数
参数名 类型 必需 描述
theme string 必需 主题名称：'light-blue' | 'light-green' | 'cream' | 'warm' | 'dark'
示例
// 切换到淡绿主题
Dancol.Theme.applyTheme('light-green');

// 切换到米黄主题
Dancol.Theme.applyTheme('cream');

Dancol.Theme.getCurrentTheme()

string getCurrentTheme()

获取当前应用的主题名称。

返回值

返回当前主题的名称字符串。

示例
const currentTheme = Dancol.Theme.getCurrentTheme();
console.log('当前主题:', currentTheme);

Dancol.Theme.getAllThemes()

Object getAllThemes()

获取所有可用主题的信息。

返回值

返回包含所有主题信息的对象。

示例
const themes = Dancol.Theme.getAllThemes();
Object.keys(themes).forEach(key => {
    console.log(`${key}: ${themes[key].name}`);
});

Toast消息

Toast消息系统提供了丰富的消息提示功能。

Dancol.Toast.show(type, message, options)

string show(type: string, message: string | Object, options?: Object)

显示Toast消息。

参数
参数名 类型 必需 描述
type string 必需 消息类型：'success' | 'warning' | 'error' | 'info'
message string | Object 必需 消息内容或配置对象
options Object 可选 额外配置选项
配置选项
选项名 类型 默认值 描述
position string 'top-right' 显示位置
duration number 4000 显示时长（毫秒），0表示不自动关闭
closable boolean true 是否显示关闭按钮
showIcon boolean true 是否显示图标
style string 'default' Toast样式：'default' | 'simple'
示例
// 基本用法
const toastId = Dancol.Toast.show('success', '操作成功！');

// 自定义配置
Dancol.Toast.show('warning', '请注意', {
    position: 'top-center',
    duration: 5000,
    closable: false
});

// 使用配置对象
Dancol.Toast.show('info', {
    title: '提示',
    message: '这是详细信息',
    position: 'bottom-right'
});

便捷方法

Toast系统提供了便捷的方法来显示不同类型的消息。

string success(message: string | Object, options?: Object)
string warning(message: string | Object, options?: Object)
string error(message: string | Object, options?: Object)
string info(message: string | Object, options?: Object)
示例

Dancol.Toast.warning('请注意这个警告');

Dancol.Toast.info('这是一条信息');

// 简洁样式便捷方法
Dancol.Toast.simpleSuccess('操作成功！');
Dancol.Toast.simpleError('发生错误');
Dancol.Toast.simpleInfo('简洁信息');

Dancol.Toast.hide(toastId)

void hide(toastId: string)

手动隐藏指定的Toast消息。

参数
参数名 类型 必需 描述
toastId string 必需 Toast的唯一标识符
示例
// 显示Toast并获取ID
const toastId = Dancol.Toast.show('info', '这条消息将在3秒后关闭', {
    duration: 0 // 不自动关闭
});

// 3秒后手动关闭
setTimeout(() => {
    Dancol.Toast.hide(toastId);
}, 3000);

Dancol.Toast.clear(position)

void clear(position?: string)

清除所有或指定位置的Toast消息。

参数
参数名 类型 必需 描述
position string 可选 要清除的位置，不传则清除所有
示例
// 清除所有Toast
Dancol.Toast.clear();

// 清除右上角的Toast
Dancol.Toast.clear('top-right');

事件系统

Dancol UI 提供了自定义事件系统，让您能够监听框架内部的状态变化。

框架事件

dancol:ready

框架初始化完成时触发。

document.addEventListener('dancol:ready', function(e) {
    console.log('Dancol UI 已准备就绪');
    console.log('框架版本:', e.detail.version);
});

主题事件

dancol:theme-changed

主题切换时触发。

document.addEventListener('dancol:theme-changed', function(e) {
    console.log('主题已切换为:', e.detail.theme);
    console.log('主题信息:', e.detail.themeInfo);
});

Toast事件

dancol:toast-show

Toast显示时触发。

document.addEventListener('dancol:toast-show', function(e) {
    console.log('Toast已显示:', e.detail.id);
    console.log('消息类型:', e.detail.type);
});

dancol:toast-hide

Toast隐藏时触发。

document.addEventListener('dancol:toast-hide', function(e) {
    console.log('Toast已隐藏:', e.detail.id);
});
