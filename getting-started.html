快速开始

几分钟内开始使用 Dancol UI Framework

安装和引入

Dancol UI Framework 是一个纯前端框架，无需复杂的构建工具，可以直接在HTML页面中引入使用。

1 下载框架文件

下载 Dancol UI Framework 的完整文件包，或者直接克隆项目到本地。

# 克隆项目
git clone https://github.com/3iXi/DancolUI.git

# 或者直接下载ZIP文件解压
2 文件结构

解压后的文件结构如下：

Dancol_UI/
├── assets/
│ ├── css/
│ │ ├── dancol-core.css # 核心样式
│ │ ├── dancol-themes.css # 主题样式
│ │ └── dancol-components.css # 组件样式
│ └── js/
│ ├── dancol-core.js # 核心功能
│ ├── dancol-theme.js # 主题切换
│ └── dancol-toast.js # Toast消息
├── examples/ # 示例页面
│ ├── admin-dashboard.html # 管理仪表板
│ ├── blog-page.html # 博客页面
│ ├── login-page.html # 登录页面
│ ├── product-showcase.html # 产品展示
│ └── user-management.html # 用户管理
├── index.html # 首页
├── api-reference.html # API文档
├── getting-started.html # 快速开始
├── examples.html # 示例页面
└── README.md # 项目说明
3 在HTML中引入

在您的HTML页面中引入必要的CSS和JavaScript文件：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的项目</title>
    
    <!-- Dancol CSS -->
    <link rel="stylesheet" href="assets/css/dancol-core.css">
    <link rel="stylesheet" href="assets/css/dancol-themes.css">
    <link rel="stylesheet" href="assets/css/dancol-components.css">
</head>
<body>
    <!-- 您的页面内容 -->
    
    <!-- Dancol JavaScript -->
    <script src="assets/js/dancol-core.js"></script>
    <script src="assets/js/dancol-theme.js"></script>
    <script src="assets/js/dancol-toast.js"></script>
</body>
</html>

基本使用

引入框架后，您就可以开始使用 Dancol UI 的各种组件和功能了。

创建第一个页面

以下是一个完整的示例页面：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Dancol UI 示例</title>
    
    <link rel="stylesheet" href="assets/css/dancol-core.css">
    <link rel="stylesheet" href="assets/css/dancol-themes.css">
    <link rel="stylesheet" href="assets/css/dancol-components.css">
</head>
<body>
    <div class="dancol-container">
        <h1>欢迎使用 Dancol UI</h1>
        <p>这是一个使用 Dancol UI Framework 创建的页面。</p>
        
        <!-- 按钮示例 -->
        <button class="dancol-btn dancol-btn-primary" onclick="showToast()">
            点击显示消息
        </button>
        
        <!-- 主题切换器 -->
        <div class="dancol-theme-switcher">
            <button class="dancol-theme-switcher-button">
                <span class="dancol-theme-color light-blue"></span>
                <span class="dancol-theme-name">淡蓝主题</span>
            </button>
            <div class="dancol-theme-switcher-dropdown"></div>
        </div>
    </div>
    
    <script src="assets/js/dancol-core.js"></script>
    <script src="assets/js/dancol-theme.js"></script>
    <script src="assets/js/dancol-toast.js"></script>
    
    <script>
        function showToast() {
            Dancol.Toast.success('Hello, Dancol UI!');
        }
    </script>
</body>
</html>
💡 提示：确保按照正确的顺序引入CSS和JavaScript文件。CSS文件应该在<head>中引入，JavaScript文件应该在</body>标签前引入。

核心概念

主题系统

Dancol UI 内置了4种主题，您可以通过以下方式切换主题：

// 切换到淡绿主题
Dancol.Theme.applyTheme('light-green');

// 获取当前主题
const currentTheme = Dancol.Theme.getCurrentTheme();

// 监听主题变更事件
document.addEventListener('dancol:theme-changed', function(e) {
    console.log('主题已切换为:', e.detail.themeInfo.name);
});

Toast消息系统

使用Toast系统显示各种类型的消息提示：

// 显示成功消息
Dancol.Toast.success('操作成功！');

// 显示错误消息
Dancol.Toast.error('发生错误，请重试。');

// 自定义配置
Dancol.Toast.show('info', '自定义消息', {
    position: 'top-center',
    duration: 5000,
    closable: true
});

布局系统

使用响应式网格系统创建布局：

<div class="dancol-container">
    <div class="dancol-row">
        <div class="dancol-col-6">左侧内容</div>
        <div class="dancol-col-6">右侧内容</div>
    </div>
</div>
